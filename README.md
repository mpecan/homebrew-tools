# homebrew-tools

Homebrew tap for [mpecan](https://github.com/mpecan) tools.

## Install

```bash
brew tap mpecan/tools
```

### Available formulas

| Formula | Description |
|---------|-------------|
| **[rippy](https://github.com/mpecan/rippy)** | Shell command safety hook for AI coding tools |
| **[tokf](https://github.com/mpecan/tokf)** | CLI output compressor for LLM context windows |

```bash
brew install rippy
brew install tokf
```

Or without tapping first:

```bash
brew install mpecan/tools/rippy
brew install mpecan/tools/tokf
```

## Formulas are auto-updated

Each formula is automatically updated by the respective project's release workflow when a new version is published.
